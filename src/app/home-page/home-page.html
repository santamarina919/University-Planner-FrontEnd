Public Graphs

Public graphs is a repository of universities across the U.S that allow anybody to create a plan to complete a degree from any of those universities. To create a plan you have to create an account and then chose a university from the support lists. From there you will see a list of degrees that are currently supported by the application. Please note that the data used to create these graphs is publicily source and thus cannot be taken as 100% correct. After planning a degree using this application it would be importatnt to double check that plan with an advisor or someone of the sort. DO NOT ASSUME THE SCHEDULER IS 100% CORRECT.
