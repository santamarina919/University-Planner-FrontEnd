Public Graphs

What is Public Graphs?

Public Graphs is a project that allows anyone to upload a degree schematic for any university. A degree schematic essentially list all the requirements of some degree at a particular school. Some requirements include required courses, prerequisites, and degree categories (The underlying data structure to represent these object and their relationships is a graph). Once created users can then create a plan that fulfills all the requirements listed in the schematic.

How can I plan using a schematic?

You can plan using a schematic by first signing up and logging in. Once logged navigate to the plans list page. There is where all the plans you already created will be located (it will be empty since you just signed up). The plans list is also where you can create plans by clicking "create a plan". From there you fill out the form and submit and you'll be redirected to the planning page.

Is the application correct?

The application uses information that is user generated. It cannot be guaranteed to be 100% correct and makes no promises of that. Instead students should look at the requirements for the specific program they are looking at and become familiar with it. Then they should return to the scheduler and use to plan a schedule while validating information and finally verifying with a counselor
